¿Qué es HTML?

HTML: Lenguaje de etiquetas de hipertexto HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript). "Hipertexto" hace referencia a los enlaces que conectan páginas web entre sí, ya sea dentro de un único sitio web o entre sitios web. Los enlaces son un aspecto fundamental de la Web. Al subir contenido a Internet y vincularlo a las páginas creadas por otras personas, te conviertes en un participante activo en la «World Wide Web» (Red Informática Mundial). HTML utiliza "marcas" para etiquetar texto, imágenes y otro contenido para mostrarlo en un navegador Web. Las marcas HTML incluyen "elementos" especiales como <head> <head>, <title>, <body>, <header>, <footer>, <article>, <section>, <p>, <div>, <span>, <img>, <aside>, <audio>, <canvas>, <datalist>, <details>, <embed>, <nav>, <output>, <progress>, <video>, <ul>, <ol>, <li> y muchos otros. Un elemento HTML se distingue de otro texto en un documento mediante "etiquetas", que consisten en el nombre del elemento rodeado por "<" y ">". El nombre de un elemento dentro de una etiqueta no distingue entre mayúsculas y minúsculas. Es decir, se puede escribir en mayúsculas, minúsculas o una mezcla. Por ejemplo, la etiqueta <title> se puede escribir como <Title>, <TITLE> o de cualquier otra forma.


Historia HTML

HTML, el lenguaje de marcado para crear páginas web, fue creado por Tim Berners-Lee en 1989 para compartir documentos científicos, aunque su primera versión oficial, HTML 1.0, se publicó en 1991. Ha evolucionado a través de varias versiones, como HTML 2.0 (1995), que estandarizó muchas características, HTML 4.01 (1999), que fue un estándar durante los 2000, y la más reciente y ampliamente utilizada, HTML5 (2014), que introdujo elementos multimedia y semánticos como video, audio, <article>, y <nav>.

Que son Atributos en html

Los atributos de HTML son palabras especiales que se añaden a la etiqueta de apertura de un elemento para proporcionar información adicional o modificar su comportamiento. Aportan funcionalidad extra, como definir el destino de un enlace (href), el texto alternativo de una imagen (alt), o una clase para aplicar estilos (class). Modifican y configuran: Ajustan cómo se ve o se comporta un elemento HTML. Sintaxis: Se escriben dentro de la etiqueta de apertura, como pares de nombre y valor (nombre="valor"), como se muestra en el ejemplo a continuación. Tipos de atributos: Existen atributos globales, que se pueden usar en cualquier elemento, y otros específicos de ciertos elementos. Ejemplo: En la etiqueta <a> el atributo href es obligatorio para que el enlace funcione correctamente.


Regla fundamentales en html

Las reglas fundamentales de HTML incluyen el uso de etiquetas que deben abrirse y cerrarse correctamente (excepto las auto-cerradas), la estructura básica del documento (<!DOCTYPE>, <html>, <head>, <body>), el anidamiento correcto de etiquetas y el uso de atributos para proporcionar información adicional. La indentación es clave para la legibilidad del código.


Etiqueta para hacer un documento html

Las etiquetas fundamentales para un documento HTML son <doctype html>, <html>, <head> y <body>. <html> contiene todo el documento, <head> contiene metadatos como el título (<title>), y <body> contiene el contenido visible de la página, como títulos (<h1>), párrafos (<p>) y enlaces (<a>). Etiquetas básicas y estructura <!DOCTYPE html>: Declara el tipo de documento y la versión de HTML. Debe estar al principio del archivo. <html>: Es el elemento raíz que envuelve todo el contenido de la página web. <head>: Contiene metadatos sobre el documento, como el título y la configuración para el navegador. No es visible directamente en la página. <title>: Define el título que aparece en la pestaña del navegador. <body>: Contiene todo el contenido visible del documento, como texto, imágenes y enlaces.
